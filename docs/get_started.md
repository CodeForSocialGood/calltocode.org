# Get Started

## 1. Install dependencies.
```bash
$ yarn
```

## 2. Setup pairing.
```bash
$ git solo <initials>                 # if you are solo
$ git duet <initials-1> <initials-2>  # if you are pairing
```

## 3. Run tests.
```bash
$ yarn test
```

## 4. Start the app.
```bash
$ yarn start:dev    # run in debug mode
$ yarn start        # run in normal mode
```
